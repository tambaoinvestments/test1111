# Logo Đối Tác

Thư mục này chứa logo của các đối tác.

## Hướng dẫn thêm logo:

1. Lưu logo với tên file theo format: `ten-doi-tac.png` hoặc `ten-doi-tac.svg`
2. Kích thước khuyến nghị: 200x200px hoặc tỷ lệ vuông
3. Format: PNG với nền trong suốt hoặc SVG
